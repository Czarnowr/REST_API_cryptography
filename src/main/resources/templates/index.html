Cryptography

REST API Documentation


About


The cryptography project is a simple REST API providing the functionality to encode and decode text using a selection of classical ciphers. The list of available ciphers and their description is available under the 'ciphers' section below.


General Information


Http method: Post
Request body: { "request" : "text to process" }
Response body: { "response" : "text after processing" }

URL


Base URL: https://czarnowr-cryptography.herokuapp.com/api
Encode text: /encode/:cipher/:modifier*
Decode text: /decode/:cipherCode/:modifier*

* Please refer to detailed cipher description below to check wherther your chosen cipher requires the use of a modifier


Ciphers


Cipher name cipher code Modifier Key
Atbash atbash - reversed alphabet
ROT 13 rot13 - alphabet moved by 13 letters
Caesar caesar n alphabet moved by n letters

Error Codes


Code Content Reason
400 BAD REQUEST { error : Input does not contain letters } No letters in request body
400 BAD REQUEST { error : Cannot process an empty String, please provide input } Empty string sent in request body

Sample Call:


Address: Http method: Body:
https://czarnowr-cryptography.herokuapp.com/api/encode/atbash POST { "request" : "Sample message" }
https://czarnowr-cryptography.herokuapp.com/api/decode/atbash POST { "request" : "hznkov nvhhztv" }
